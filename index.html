fukuii

A Scala-based client for Ethereum-like blockchains

fukuii is a robust Ethereum Classic client forked from IOHK's mantis project, [github](https://github.com/chippr-robotics) providing production-ready blockchain infrastructure with comprehensive tooling for developers and organizations.

Download Get Started View on GitHub

Key features and capabilities

fukuii inherits powerful features from the mantis project, delivering a complete blockchain client implementation optimized for Ethereum Classic and compatible networks.

üîó

Full blockchain client

Complete implementation supporting Ethereum Classic, Ethereum, Mordor testnet, and internal testnets with latest hard-fork compatibility including Magneto (ECIP-1103). [GitHub](https://github.com/input-output-hk/mantis)

üîå

JSON-RPC API

Industry-standard JSON-RPC interface for seamless blockchain interaction with HTTP/HTTPS support and optional TLS encryption for secure communications. [github](https://github.com/input-output-hk/mantis)

üíº

Wallet management

Built-in wallet functionality with account management, compatible with standard Ethereum keystore format, and password-protected key encryption.

‚öôÔ∏è

Comprehensive CLI tools

Command-line interface for private key generation, address derivation, genesis allocations, key-pair generation, key encryption, and keystore management. [github](https://github.com/input-output-hk/mantis)

üåê

P2P networking

Robust peer-to-peer networking via devp2p protocol with configurable ports, ensuring reliable connection to Ethereum Classic and compatible networks.

üö∞

Testnet faucet

Built-in faucet functionality for automated testnet fund distribution, making it easy to provide gas for development and testing accounts. [github](https://github.com/input-output-hk/mantis)

üìä

Monitoring & metrics

Integrated Prometheus metrics and pre-configured Grafana dashboards for comprehensive performance monitoring and operational visibility. [github](https://github.com/input-output-hk/mantis)

üîí

Enterprise-grade crypto

Production-ready cryptographic implementations including secp256k1, Keccak-256, RIPEMD-160, SHA-256, [GitHub](https://github.com/input-output-hk/mantis/blob/develop/crypto/src/main/scala/io/iohk/ethereum/crypto/package.scala) and Ethereum IES encryption.

üê≥

Flexible deployment

Multiple deployment options via source builds, Nix packages, or Docker containers with docker-compose configurations for easy setup. [github](https://github.com/input-output-hk/mantis)

Getting started with fukuii

Comprehensive guides to help you install, configure, and use fukuii effectively.

üöÄ Quick start

Starting the client

Launch fukuii connected to your preferred network:

./bin/mantis-launcher [network]

Supported networks:

  • etc - Ethereum Classic mainnet [github](https://github.com/input-output-hk/mantis)
  • eth - Ethereum mainnet [github](https://github.com/input-output-hk/mantis)
  • mordor - Mordor testnet
  • testnet-internal - Internal testnet [github](https://github.com/input-output-hk/mantis)

üîë CLI tools

Generate a private key

./bin/mantis cli generate-private-key

Derive an address

./bin/mantis cli derive-address [private-key]

Encrypt a private key

./bin/mantis cli encrypt-key \
--passphrase=[password] [private-key]

Generate multiple key-pairs

./bin/mantis cli generate-key-pairs 5

üåê RPC API usage

Create a new account

curl --request POST \
--url http://127.0.0.1:8546/ \
--header 'Content-Type: application/json' \
--data '{
"jsonrpc": "2.0",
"method": "personal_newAccount",
"params": ["<password>"],
"id": 1
}'

üìä Monitoring setup

Enable metrics

Configure metrics in metrics.conf:

mantis.metrics.enabled=true

Start monitoring stack

./docker/mantis/build.sh

Access Grafana at http://localhost:3000 (credentials: admin/admin) [github](https://github.com/input-output-hk/mantis)

üîí TLS/HTTPS configuration

Generate certificates

./tls/gen-cert.sh

Configure HTTPS mode

Set in application.conf:

mantis.network.rpc.http.mode=https

‚ö° Development mode

Fast incremental compilation

Enable development mode for faster builds:

export MANTIS_DEV=true

Or use system property:

-DmantisDev=true

About fukuii's origins

fukuii is a fork of the mantis project, originally developed by IOHK (Input Output Hong Kong) for the Ethereum Classic network. [github](https://github.com/chippr-robotics) The mantis project was created to provide a robust, production-ready Ethereum Classic client implementation in Scala. [GitHub](https://github.com/input-output-hk/mantis)

Chippr Robotics maintains the fukuii fork as part of their commitment to advancing blockchain technology and supporting the Ethereum Classic ecosystem. The organization focuses on automation, AI, and blockchain technologies with a mission of "freeing humanity from labor through automation." [Chipprbots](https://chipprbots.com/about/)

Key milestones

  • Original mantis development by IOHK for Ethereum Classic
  • Magneto hard-fork support (ECIP-1103) integration [GitHub](https://github.com/input-output-hk/mantis)
  • fukuii fork by Chippr Robotics for continued development
  • Ongoing enhancements to blockchain infrastructure
